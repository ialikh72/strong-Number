Strong Number Program in C#
Description

This is a C# console program to find strong numbers within a given range.

A Strong Number is a number such that the sum of the factorial of its digits equals the number itself.

Example:

1 → 1! = 1 ✅

2 → 2! = 2 ✅

145 → 1! + 4! + 5! = 145 ✅

The program uses loops, arrays, and a function to calculate factorial of digits.

Features

Takes user input as the upper limit of numbers to check.

Loops from 1 to the input number.

Calculates sum of factorial of each digit for every number.

Prints all strong numbers found in the range.
